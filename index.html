SAMP Extension Examples

Following examples have been copied from http://astrojs.github.io/sampjs and adapted in order to work with the dedicated browser extension.

You need to install the extension on your browser before using them. See repo README: https://github.com/zonia3000/samp-browser-extension for more information.

  • Pinger: really really minimal SAMP web page
  • Sender: very simple script to send a table
  • Table Viewer: receives a table from another client and does 2-way communications
